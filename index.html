Game options

  1. New Game
    2. - Restart game with the current settings.
  2. Start Game
    3. - Monsters will start to attack.
  3. Gacha
    4. - This will activate a slot machine to randomly select from a list of searching algorithms.
    - The AI will scan the surroundings to find the shortest path towards the exit.
    - After scanning, a lighted path will be displayed.
    - All monsters will be deactivated during and after this process.
    - This can only be used once.
  4. Maze Type
    5. - Choose between recursively generated maze or randomly generated maze.
  5. Game mode
    6. - Choose between having walls as obstacles or bombs as obstacles.
    - Unlike Bombs, Walls cannot be penetrated.
  6. Voice Mode
    7. - Use Speech Recognition to activate commads

Tutorial

  1. The goal of this game is to control your player and get to the exit with the least number of moves.
  2. Control player via wasd or arrow keys.
    3. Player:  Exit:  Wall:  Bomb:  Explosion:  Lighted Path: 
  3. Moves are counted by the number of times you pressed the keyboard.
  4. If you touched a bomb, or explosion, 1 move will be multiplied by 5.
  5. Your character will complain to you if you run into a wall, bomb or explosion.
  6. No hearts will be lost if you run over a bomb.
  7. Monsters will search for your location once and will move towards and until it reaches the end of the searched path, before searching for your position again.
    8. Monster: 
  8. The number of monsters are random, however, the max number will vary according to the map size.
  9. Each time the monster touches you within it's path, 1 heart will be lost.
    10. Heart: 
  10. Lose 3 hearts and you will have 1 life left before losing the game.
  11. One way of avoiding the monsters is to move away from the monster's path, while the monster is currently moving.
  12. Ambushing a monster, while not being in the monster's path will allow you to kill the monster temporarily without losing a heart. However, monster will revive at it's death location almost immediately.
  13. The only way to permanently kill a monster is to kill it at it's last path, which is at your location if you did not move. However, doing so will cost you 1 heart.

Voice Commands

  1. Move up - move up by 1 step
  2. Move right - move right by 1 step
  3. Move down - move down by 1 step
  4. Maze left - move left by 1 step
  5. Up end - move up till the last obstacle
  6. Right end - move right till the last obstacle
  7. Bottom end - move bottom till the last obstacle
  8. Left end - move bottom till the last obstacle
  9. New game - play a new game with the same settings
  10. Start game - enable monsters to start attacking
  11. Help - activate GACHA to roll for a search algorithm to find the shortest path to the exit
  12. End game - automatically move player to the exit, will not work if help command or GACHA was not clicked prior to this command
  13. Activate recursive maze - enable recursive backtracking algorithm type maze
  14. Activate random maze - enable random type maze (default)
  15. Activate bombs mode - enable bomb type maze, can penetrate
  16. Activate maze mode - enable wall type maze, cannot penetrate
  17. Deactivate voice mode - disable voice mode
